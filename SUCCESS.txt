🚀 QUICK START
═══════════════════════════════════════════════════════════════════════════════

  1. Install Dependencies
     $ pnpm install

  2. Build All Packages
     $ pnpm build

  3. Run Backend (MCP Server)
     $ cd packages/backend
     $ pnpm dev

  4. Run Client (Claude Integration)
     $ cd packages/client
     $ export ANTHROPIC_API_KEY=sk-your-key
     $ pnpm dev

  5. Run Frontend (Chat UI)
     $ cd packages/frontend
     $ pnpm dev
     # Opens on http://localhost:3000
